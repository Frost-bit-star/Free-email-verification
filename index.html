HOW TO EMBED STACKVERIFY HTML + JS

📋 Step 1: Place the input fields inside your existing form where users will register. These fields will be used to collect the user's basic info before verifying their email.

🔐 Step 2: When the user clicks Send OTP, a 6-digit code will be emailed to them using StackVerify. They’ll enter it to verify.

📤 Step 3: Once OTP is verified, StackVerify automatically submits their data to your API (defined in data-register-endpoint).

➡️ Step 4: After registration, the user will be redirected to the URL set in data-redirect.

🔧 Tip: Customize the API endpoint and redirect path for each platform using script data-* attributes.
